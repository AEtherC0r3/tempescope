Tempescope Quickstart

簡単にTempescope SDKの使い方を解説します。
Tempescopeは現在Javaによる制御が可能です。

環境構築

1. tempescope.jarをダウンロードし、クラスパスに置く
2. USB-TTLドライバをインストール
3. (OSXの場合）librxtxSerial.jnilibを/Library/Java/Extensions/に置く
4. TempescopeリモコンをPCに接続し、Tempescope本体が近くにあり、電源が入っていることを確認してください。

例1：好きな天気を再生

Tempescopeに「夕方」の「晴れ」を表示するデモです。
import com.tempescope.controller.TempescopeController;
import com.tempescope.model.Weather;
import com.tempescope.model.WeatherType;

public class PlayWeather
{
	
	public static void main(String[] args)
	{
		TempescopeController tempescope=new TempescopeController();
		
		double sunLevel=0.6;
		WeatherType weatherType=WeatherType.kClear;
		boolean lightning=false;
		
		tempescope.playWeather(new Weather(sunLevel, weatherType,lightning));
		
	}
}
1行ずつ見ていきましょう。
まずTempescopeControllerオブジェクトを作ります。これは、USBで接続したTempescopeリモコンへの接続を行います。 
TempescopeController tempescope=new TempescopeController();

天気をtempescopeに送ります。
sunLevel： 0から1で太陽の色を表します。0=暗闇 0.5=朝焼け/夕焼け 1=昼
weatherType： 晴れ（kClear）、雨（kRain）、曇（kCloud）
lightning： trueの場合雷が出ます。 
		double sunLevel=0.6;
		WeatherType weatherType=WeatherType.kClear;
		boolean lightning=false;
		
		tempescope.playWeather(new Weather(sunLevel, weatherType,lightning));

Tempescope Stream API

よりローレベルに制御したい人は、315MHzRFで直接シリアル通信してください：

天気を再生する

フォーマット： 
Zr,[color],[weather],[lightning]
color= 0~100　0=夜　0.5=朝焼け　1=昼
weather= 0:晴れ　1:雨　2:曇り
lightning= 0:雷なし　1:雷あり

例１：　昼に雨が降ってる
Zr,100,1,0

例２：　早朝に曇ってる
Zr,23,2,0

例３：　夜中に雷雨
Zr,0,1,1
